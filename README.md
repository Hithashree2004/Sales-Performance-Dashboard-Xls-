# 📊 Sales Performance Dashboard (Excel)

## 📌 Overview

This project presents an interactive Sales Performance Dashboard built using Microsoft Excel to analyze key business metrics such as revenue, profit, and product performance. The dashboard transforms raw data into actionable insights for better decision-making.

---

## 🎯 Objectives

* Evaluate overall sales and profit performance
* Identify top-performing regions and products
* Analyze trends over time
* Enable dynamic and interactive data exploration

---

## 🛠 Tools & Technologies

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Slicers (Interactive Filters)
* Data Cleaning & Preparation

---

## ⚙️ Key Features

* 📌 KPI Metrics: Total Sales, Total Profit
* 📊 Time-based Sales Trend Analysis
* 🌍 Region-wise Performance Insights
* 🛍 Product-wise Sales Analysis
* 🎛 Interactive Dashboard with Slicers
